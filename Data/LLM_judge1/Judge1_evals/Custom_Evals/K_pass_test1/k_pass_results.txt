WHAT IS A K-PASS TEST?
----------------------------------------------------------------------
  A K-pass test evaluates each answer 5 times, each time from
  a different evaluative perspective. An answer must hold up
  under ALL K passes to be considered truly reliable.

  Pass threshold: score >= 0.5 to PASS a pass.
  Weighted K-pass score = weighted mean across all passes.

  Pass 1 -- Factual Pass (weight=30%)
    Are stated numbers correct vs ground truth? (tolerance 5%)

  Pass 2 -- Completeness Pass (weight=25%)
    Does prediction cover all key GT concepts?

  Pass 3 -- Semantic Pass (weight=20%)
    Do prediction and GT express the same meaning?

  Pass 4 -- Terminology Pass (weight=15%)
    Does prediction use correct biomedical terminology?

  Pass 5 -- Consistency Pass (weight=10%)
    Is the prediction internally self-consistent?
