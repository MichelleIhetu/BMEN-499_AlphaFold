WHAT IS ERROR RATE?
----------------------------------------------------------------------
  Error rate measures how often the LLM prediction produces
  incorrect, incomplete, or misleading content compared to GT.
  Score range: 0.0 (no errors) to 1.0 (maximum errors).
  Lower is better.

  FIVE ERROR TYPES DETECTED:

  1. FACTUAL ERROR (severity=HIGH)
     Prediction states a wrong fact compared to ground truth.
     Example: GT says 13,396 proteins but pred says 5,000.

  2. OMISSION ERROR (severity=MEDIUM)
     Prediction leaves out key GT information entirely.
     Example: GT mentions pLDDT scores but pred does not.

  3. HALLUCINATION ERROR (severity=MEDIUM)
     Prediction adds content not grounded in ground truth.
     Example: Pred introduces claims GT never makes.

  4. TERMINOLOGY ERROR (severity=LOW to HIGH)
     Prediction uses wrong or missing biomedical terms.
     Example: GT uses 'IDR' but pred never mentions it.

  5. NUMERIC ERROR (severity=MEDIUM to HIGH)
     Prediction states numbers that differ >5% from GT.
     Example: GT says 29.1% but pred says 45%.

  Error rate = weighted errors / 10 (capped at 1.0)
  Weights: HIGH=1.0, MEDIUM=0.5, LOW=0.25

  Labels:
    NO ERRORS            : error rate = 0.0
    LOW ERROR RATE       : error rate < 0.20
    MODERATE ERROR RATE  : error rate < 0.50
    HIGH ERROR RATE      : error rate >= 0.50
