METHOD OVERVIEW
----------------------------------------------------------------------
  BERTScore measures semantic similarity between a candidate
  answer and a reference answer using three metrics:
    Precision (P) -- candidate tokens grounded in reference
    Recall    (R) -- reference tokens covered by candidate
    F1            -- harmonic mean of P and R

  Reference answers are drawn from the DisProt KB passages
  (KB-001 through KB-014) matched to each question by the
  top retrieved document ID in the prediction file.

INTERPRETATION
----------------------------------------------------------------------
  High F1 scores indicate the predicted answer closely mirrors
  the reference KB passage -- expected in extractive fallback
  mode since answers ARE concatenated KB passages.

  Questions with low F1 indicate retrieval mismatch: the top
  retrieved document does not well represent the question, so
  the reference-answer semantic overlap is low even though the
  answer itself may be factually correct on a different topic.

  For true BERTScore with contextual embeddings, install:
    pip install bert-score
  and replace the TF-IDF vectors with actual BERT embeddings.
