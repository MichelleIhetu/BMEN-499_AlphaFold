# BMEN-499_AlphaFold
Independent Research Project

The goal of this course is to create Question Answer (QA) baseline then compare it to the ground truth answers to 3 LLMs predicted answers. 

BioGPT model will be used as the baseline model to evaluate 100 generated questions  from ChatGPT-5.3 to create ground truth answers. 

Each LLM will generate their own respectieve predicted answers which will then be compared to the baseline LLM

LLM-1: BiomedBERT + Cailbrated Symbolic rules 

LLM-2: BiomediBERT

LLM-3:BiomedBERT + Biomistral
